# py-dpasp
A Pure Python Implementation of the DPASP System (For Differential Probabilistic Answer Set Programming)

Install the five python programs in the py-dpasp-solver folder and all the requirements in requirements.md

To solve a dPASP program, simply enter in your terminal:
```bash
python3 solver.py your_program 
```

OBS: The current implementation assumes a ground program without Probabilistic rules with body.
